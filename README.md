# epoll_server

#### 介绍
epoll_server 是使用C++ 基于 linux 的 epoll 模块开发的一个高性能tcp server, 使用面向对象的方法对 epoll 函数 做了基础包装

#### 软件架构
软件架构说明


#### 安装教程

1.  mkdir build && cd build
2.  cmake ..
3.  make

#### 使用说明

1.  必须linux系统下编译运行且内核版本>=2.6
2.  g++ 支持 C++11  (gcc版本>=4.7)
